This is the first post on Blogo.

The site now uses the Minimal Mistakes theme, so posts render with its `single` post layout and the homepage uses the theme's blog index.

Add new posts in the `_posts/` directory, push to `main`, and GitHub Actions will deploy the updated site to GitHub Pages.
